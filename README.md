# 🛡 СВОЙ — Telegram-бот защиты от мошенников

**Версия:** 0.1.0 (MVP)

Бот анализирует подозрительные сообщения, ссылки и номера телефонов.
Пользователь пересылает боту текст — бот отвечает: опасно или нет.

---

## Что умеет

- **Анализ текста** — определяет маркеры манипуляции (срочность, страх, давление, подмена авторитета)
- **Проверка ссылок** — WHOIS домена (возраст, владелец), сравнение с известными брендами
- **Проверка номеров** — поиск по базе известных мошеннических номеров
- **Статистика** — сколько угроз обнаружено

---

## Установка и запуск

### 1. Создай бота в Telegram

- Открой [@BotFather](https://t.me/BotFather)
- Отправь `/newbot`
- Придумай имя (например: `СВОЙ Защита`) и username (например: `svoy_protect_bot`)
- Скопируй токен

### 2. Установи зависимости

```bash
cd svoy_bot
python3 -m venv venv
source venv/bin/activate      # Mac/Linux
# venv\Scripts\activate       # Windows
pip install -r requirements.txt
```

### 3. Настрой конфиг

```bash
cp .env.example .env
```

Открой `.env` и вставь токен бота:

```
BOT_TOKEN=твой_токен_от_BotFather
```

### 4. Запуск

**Основной способ (с Docker)**
Запускает бота (вместе с OSINT агентами) и веб-панель администратора в изолированных контейнерах:

```bash
docker-compose up --build -d
```

Панель администратора будет доступна по адресу `http://localhost:5005`.

**Альтернативный способ (Локально без Docker)**

```bash
python bot.py
```

И в другом терминале:

```bash
python admin_panel.py
```

Готово! Открой бота в Telegram и отправь подозрительное сообщение.
