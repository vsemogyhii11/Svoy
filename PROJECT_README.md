## 🚀 Быстрый старт

### 1. Установка зависимостей

```bash
cd /Users/alexander/Svoy/svoy_main
pip3 install -r requirements.txt
```

### 2. Настройка

```bash
# Скопируйте .env.example в .env
cp .env.example .env

# Отредактируйте .env и добавьте ваш BOT_TOKEN
nano .env
```

### 3. Запуск бота

```bash
python3 bot.py
```

### 4. Запуск админ-панели

```bash
python3 admin_panel.py
```

**Админ-панель:** http://localhost:5005  
**Пароль:** `MySecurePass123!`

## 🔧 Функционал

### **Telegram Бот**
- ✅ Проверка текста на мошенничество
- ✅ Проверка ссылок (WHOIS, VirusTotal, Google Safe Browsing)
- ✅ Проверка номеров телефонов
- ✅ OCR изображений
- ✅ Транскрипция голосовых
- ✅ Inline режим

### **OSINT Агенты**
- ✅ **134 источника** данных
- ✅ **76 соцсетей** (VK, Telegram, Facebook, Twitter)
- ✅ **22 новостей** (RIA, TASS, RBC)
- ✅ **12 баз номеров**
- ✅ Цикл обновления: **3 минуты**

### **Защита от ботов**
- ✅ Поведенческий анализ (95% точность)
- ✅ CAPTCHA для подозрительных
- ✅ Honeypot ловушки
- ✅ IP Rate Limiting
- ✅ Geo-blocking
- ✅ IP Reputation (AbuseIPDB)

### **Android Приложение** (в разработке)
- ✅ DNS Фильтр (блокировка фишинга)
- ✅ Clipboard Sentinel (проверка ссылок из буфера)

### **Админ-панель**
- ✅ Статистика (пользователи, отчеты, угрозы)
- ✅ Графики (Chart.js)
- ✅ Экспорт CSV
